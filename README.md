# Carrinho_Compras_React
Componente de Carrinho de compras onde os produto também são adicionados via página principal. Na área de produtos apresentada. O componente de navegação devera possuir um link para acesso do Carrinho. Ao acessar o carrinho deverá ser possível excluir produtos. Se o Carrinho estiver vazio deverá aparecer uma mensagem "Seu carrinho está vazio".
